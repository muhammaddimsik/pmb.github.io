Maaf Ijazah yang anda unggah tidak terdaftar di kemendikbud

  • Seusuaikan nama file yang anda unggah dengan format 'NoPendaftaran-NamaFile', contoh: 'PMB10233-ijazah.pdf'

  • Pastikan untuk mengisi semua data biodata diri, data keluarga, dan pilihan prodi sebelum menekan tombol kirim di bawah ini.
  • Setelah menekan tombol kirim, pastikan untuk mengecek halaman ini secara berkala dan menunggu petugas melakukan validasi data Anda.
